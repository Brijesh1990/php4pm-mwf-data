    SQL Join :

    sql join is used to join more than one tables with common field or if data matched one table with another table it will be joined

    Types of sql join

    1) join
    2) inner join
    3) outer join
    types of outer join
    a)left outer join
    b)right outer join
    c)full outer join
    4) cross join


    1) join : join all matched common data examples : select tbl_users.*,countryname,statename,ctyname from tbl_users join tbl_country on tbl_users.countryid=tbl_country.countryid join tbl_state on tbl_users.stateid=tbl_state.stateid join tbl_city on tbl_users.cityid=tbl_city.cityid;

    2) inner : join all macthed common data examples : select tbl_users.*,countryname,statename,ctyname from tbl_users inner join tbl_country on tbl_users.countryid=tbl_country.countryid inner join tbl_state on tbl_users.stateid=tbl_state.stateid inner join tbl_city on tbl_users.cityid=tbl_city.cityid;

    3) left join : left join are used to matched first table of left rows with second tables of left rows if data are matched join all data. examples : select tbl_users.*,countryname,statename,ctyname from tbl_users left join tbl_country on tbl_users.countryid=tbl_country.countryid left join tbl_state on tbl_users.stateid=tbl_state.stateid left join tbl_city on tbl_users.cityid=tbl_city.cityid;

    3) right join : right join are used to matched second table of right rows with first tables of right rows if data are matched join all data otherwise return null values. examples : select tbl_users.*,countryname,statename,ctyname from tbl_users left join tbl_country on tbl_users.countryid=tbl_country.countryid left join tbl_state on tbl_users.stateid=tbl_state.stateid left join tbl_city on tbl_users.cityid=tbl_city.cityid;

    4) full join : left jopin + right join but mysql is not supported

    5)cross join : cross of tables rows itself and retuirn dublicate join data examples : select * from tbl_users cross join tbl_country;
